# UnityCalculator

## 使い方
`Assets/Editor/CalculatorWindow.cs`だけ取り出して，自身のプロジェクトの任意の場所の`**/Editor`ディレクトリに入れるだけです．

メニューバーの`Tool/Calculator`からWindowの以下の電卓のWindowが生成できます．
